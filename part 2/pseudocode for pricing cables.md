*We will assume that the user needs extra cable and the product price will be added to the final cable price*

**CL = Cble Length**

**CP = Cble Price**

**PV = Pump Voltage**

1. user selected a pump and needs cable
2. if PV = 24V
    - if CL > 50
        - CP = 0 (not available)
    - if CL > 25 and < 50
        - CP = CL * $6
    - else
        - CP = CL * $4.8
3. if PV = 48V
    - if CL > 70
        - CP = 0 (not available)
    - if CL > 35 and < 70
        - CP = CL * $6
    - else
        - CP = CL * $4.8
4. if PV = 72V
    - if CL > 80
        - CP = 0 (not available )
    - if CL > 40 and < 80
        - CP = CL * $6
    - else
        - CP = CL * $4.8
